Bisection Method

About

The bisection method is a numerical technique used to find the roots of a continuous function. It is a simple and robust method, which repeatedly bisects an interval and then selects a subinterval in which a root must lie for further processing. This method is guaranteed to converge to a root as long as the function changes sign over the interval.

Steps to Calculate Using Bisection Method

  • Choose an initial interval [a, b] such that f(a) and f(b) have opposite signs (i.e., f(a) * f(b) < 0). This implies there is at least one root in the interval.
  • Compute the midpoint of the interval, \( c = \frac{a + b}{2} \).
  • Evaluate the function at the midpoint, f(c).
  • Determine the subinterval [a, c] or [c, b] in which the root lies:
    • If f(a) * f(c) < 0, then the root lies in [a, c]. Set b=c.
    • If f(b) * f(c) < 0, then the root lies in [c, b]. Set a=c.
  • Repeat the process until the interval is sufficiently small or until the value of f(c) is close enough to zero.

Formula

The formula for the midpoint \( c \) is given by:

c = \(\frac{a + b}{2}\)

Example

Suppose we want to find a root of the function \( f(x) = x^3 - x - 2 \) in the interval [1, 2].

  • Step 1: Compute the midpoint, \( c = \frac{1 + 2}{2} = 1.5 \).
  • Step 2: Evaluate the function at the midpoint, \( f(1.5) = 1.5^3 - 1.5 - 2 = -0.125 \).
  • Step 3: Since f(1) * f(1.5) < 0, the root lies in [1, 1.5]. Set b=1.5.
  • Step 4: Repeat the steps until the interval is sufficiently small.

Advantages

  • Guaranteed to converge if the initial interval is chosen correctly.
  • Simple to understand and implement.

Disadvantages

  • Converges slowly compared to other methods like Newton-Raphson.
  • Requires the function to be continuous in the interval.
